A collaborative platform where structure, style, and content are fully separated. Share templates, clone into projects, and fill content manually or via AI.
---------------



🌟 Key Features
---------------

* **Template Management**: Create, share, and discover reusable structural templates
* **Project Editor**: Visual editor for content and CSS variables without touching code
* **AI Integration**: Generate content, styles, and full templates using AI
* **Social Features**: Upvote templates and discuss via comments
* **Multi-language**: Full RTL support for Arabic and other languages
